# CHRONO LABELER: dispositivo fechador de alimentos para mini impresoras Bluetooth.
Este proyecto permite generar etiquetas con fecha y hora utilizando un Arduino ESP32 y una mini impresora Bluetooth; sin necesidad de utilizar un celular o una PC.
